If you do want a webpage:

  1. Add an index.md or index.html file with valid HTML/Markdown content.

  2. Remove or exclude your Python code from the Pages build by adding a .nojekyll file in the root of your repo.

  3. Commit & push changes.

  4. Re-run the build.
